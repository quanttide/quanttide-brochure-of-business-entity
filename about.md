# 公司简介

# 一、公司概览
- 量潮科技（Quanttide）是一家成立于2018年的数据科技公司，专注于为高校科研机构与企业客户提供数据采集，数据清洗处理，文本数据分析等全链路数据解决方案与服务。
- 浙江理工大学大数据微专业的官方企业授课单位，深度参与《数据科学》《数据工程》《大数据项目实践》等核心课程的设计与教学。
- 我们的名字“量潮”寓意着“量化浪潮”，代表了我们以可被量化的准确数据来驱动决策、以技术赋能行业浪潮行进的核心理念。
# 二、核心业务板块
- 科研服务：为高校研究者、科研团队、创业者及创业团队提供数据采集与清洗、数据软件产品设计与开发、数据采集与分析系统解决方案等外包服务，解决技术问题。
- 数据咨询:为学术研究领域以及有专业需求的企业部门建立数据库，爬虫获取学术网站公开数据，进行专业的数据采集、标注和清洗，为相关的单位建立结构清晰可靠的数据库。
- 课堂服务：面向高校学生和行业新手，提供数据科学类编程培训班、大数据和人工智能类科普讲座，帮助学员获得大数据领域基础知识与实践技能。
# 三、核心竞争力
- 人力资源：背靠高校，有充足的高校学生人才为项目解决提供多元化的解决想法与足量的帮助。
- 经验：迄今为止，公司已完成相关方向的十余个完整项目，有充足的项目经验，历史项目经验的完整记录与总结归纳。
- 阵容：公司内技术方向的员工及顾问来自各大海内外高校，有过硬的能力背书与充足的项目经验。
