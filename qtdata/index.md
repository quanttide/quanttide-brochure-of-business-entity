# 量潮数据


## 我们的服务是什么？

设想有个经济学家组成的科研团队正在开启一个关于技术创新的经济学研究课题。他们需要使用来自某个群体的GitHub历史提交数据作为他们参与技术创新的证据。

研究团队有一个通过前期工作积攒下来的GitHub账号名称，并且找到了一个专门提供GitHub的网站。但是这个原始GitHub数据每个压缩包就有几百个G。研究团队希望在一个星期左右就可以获得清洗以后的数据。

我们接到这个需求以后，评估大概需要开发一些Python脚本，并且把程序部署到海外区的云服务器上运行。于是我们根据开发和部署的定价规则给客户一个报价，客户觉得没问题，我们收了首付款就开始执行。一般都可以在交付时间完成。然后这一单就做完了，我们就得到了尾款。

类似地，我们接各类数据采集和数据预处理任务，科研工作者自己完成后续的数据分析和可视化工作。


## 服务核心

量潮科技专注于为国内外高校经管专业科研工作者提供"科研级"数据处理服务。我们的客户通常科研经费充足但发表压力大，需要在三年内发表多篇顶刊论文。我们将您从非核心的数据处理工作中解放出来，让您更专注于研究本身。

我们的服务覆盖数据采集与数据预处理全流程，全程严格遵循科研领域对数据准确性、可靠性的高标准要求。接到您的需求后，我们会先评估需求适配性并制定专属数据处理技术方案。确认后，会依据开发难度与部署规模的定价规则为您出具详细报价。双方确认合作并完成首付款支付后，我们将正式启动项目执行，确保在约定交付周期内完成全部服务。
